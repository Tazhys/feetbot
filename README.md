# FeetBot

A Discord bot that generates and sends AI feet pictures using Google's Gemini API when commanded.

**Commands:**
- `pls feet` - Get an AI-generated feet picture
- `pls stop` - Stop the bot from responding to feet picture requests

This bot was requested by a weird member of the ntts server ([discord.gg/ntts](https://discord.gg/ntts)), and I gave in because I thought it would be funny.

Pull requests are allowed!
